Creating tooltip which can be resuable

Code Example

```
    <Tooltip
    	position="bottom"
    	content="<< TOOLTIP TEXT >>"
    >
    	<< YOUR COMPONENT >>
    </Tooltip>
	{/* possible value of prop "position" is "bottom" or "top" (as of now) */}
```
